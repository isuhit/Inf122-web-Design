About the Course

This is a brief introduction to the web programming course. Web programming involves creating interactive and dynamic websites using languages like HTML, CSS, and JavaScript that can work on standalone systems or the internet.

Course Modules

The course is divided into three modules, each focusing on a different aspect of web programming. Each module includes lectures, assignments, and projects to help you learn and apply the concepts covered.

Module 1: Web programming basics

This module begins by introducing some theoretical concepts of the internet and the web. It explains the importance and need of web programming.

Module 2: Hypertext Markup Language

This module covers the basics of HTML, including how to structure a web page, use tags and attributes, and create links and forms.

Module 3: Cascading Style Sheets

This module focuses on CSS, which is used to style and layout web pages. It covers topics such as selectors, properties, and responsive design.
